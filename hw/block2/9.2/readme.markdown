# Task 9.2 (Reentrant TAS)

## Description

Implement reentrant test-and-set lock using `java.util.concurrent.AtomicStampedReference` and `Thread.currentThread`.
Provide at least 3 tests written in JCStress.

## Requirements

Provide source code for your class and tests, document key parts of your solution with javadoc, explain why your solution is correct in several sentences.
